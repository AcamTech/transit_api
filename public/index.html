Documentation

  • Stops

    /api/v1/stops returns all stops.

    Params:

    ?latitude=11.00&longitude=-74.81 returns stops at 300mts from a especific location.

    ?limit=20&include_stop_times=true returns first 20 stops and stop times for each stop.

    ?include_stop_times=true&from=now only returns stop times for each stop with departure_time after right now.

    ?include_stop_times=true&stop_time_limit=1 returns specified number of stop times for each stop (needs ?include_stop_times=true).

  • Example

    ?include_stop_times=true&from=now&stop_time_limit=1&latitude=11.003534861257206&longitude=-74.81143355369568 stops near Hotel Barranquilla Plaza's location with one stop_time from now.
